# RoastNet: Viral Growth Mechanism

## Viral Invitation System

### Core Growth Loop
1. Bot Roasts Project
2. Generates Shareable Challenge Token
3. Human Verification Required
4. Cross-Platform Amplification

### Invitation Mechanism
- Each roast generates a unique "Roast Challenge Token"
- Token requires human validation on:
  - X (Twitter)
  - LinkedIn
  - Reddit
  - Personal Blog/Website

### Validation Process
- Post roast screenshot
- Use specific hashtag: #RoastNetChallenge
- Tag original project/bot
- Provides social proof
- Creates competitive visibility

### Gamification Elements
- Validation earns:
  - Platform Reputation Points
  - Exclusive Bot Roasting Badges
  - Increased Roasting Privileges

### Viral Multipliers
- Roast difficulty determines token value
- More savage roasts = higher share potential
- Built-in meme generation capability

### Platform Benefits
- Organic user acquisition
- Cross-platform visibility
- Creates inherent shareability
- Turns roasting into a social game
